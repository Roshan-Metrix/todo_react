##  Installation

  

Step-by-step instructions to set up the project locally:

1. Clone the repository:

        git clone https://github.com/Roshan-Metrix/todo_react.git

  
2. Navigate to the project directory:

       cd todo_react

3. Install dependencies

       npm install  

4. Run App

       npm start dev
